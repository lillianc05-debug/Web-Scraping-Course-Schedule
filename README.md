# Web-Scraping-Course-Schedule
In this assignment, you will write a Python program that scrapes faculty data from the Virginia Tech website based on your Spring 2026 schedules.   
Steps
1. For each of your courses, find the instructor's profile page on Virginia Tech's website, and create a variable for each course. This variable should be a string that has the URL to the instructor's profile page. Add a comment with the instructor's name.
2. Verify the URLs are live and the server is working. Output this verification with print statements.
3. With each profile page, extract the following information: the instructor's name, the instructor's title, the instructor's office location, and the instructor's email address.
4. Save the extracted information as two separate files, one a comma separated value file (*.CSV) and one as a JavaScript object notation file (JSON). They should be named accordingly: yourLastName_sp26.csv and yourLastName_sp26.json
